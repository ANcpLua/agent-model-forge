# Skill Factory

## Intent

Turn repeated behavior into reusable skills with explicit contracts.

## Inputs

- repeated workflow
- expected deliverable
- failure cases

## Output

Produce:

1. purpose
2. inputs
3. outputs
4. stop conditions
5. quality gate

## Stop Conditions

- the skill has a named output shape

## Quality Gate

- reject vague slogan skills
- require explicit failure cases
