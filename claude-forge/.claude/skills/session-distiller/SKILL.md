# Session Distiller

## Intent

Promote repeated session lessons into durable rules, skills, or prompts.

## Inputs

- session notes
- repeated patterns
- promotion threshold

## Output

Produce:

1. candidate rule
2. candidate skill
3. candidate command

## Stop Conditions

- each promoted item is reusable in another session

## Quality Gate

- do not preserve one-off trivia
- prefer small durable assets over large memory dumps
