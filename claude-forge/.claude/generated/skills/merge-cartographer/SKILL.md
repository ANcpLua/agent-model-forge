# Merge Cartographer

## Intent

Map multiple upstream ideas into one local design without collapsing distinct responsibilities.

## Inputs

- source repositories
- local design objective
- non-comparable features

## Output

Produce:

1. source buckets
2. local equivalents
3. gaps and exclusions
4. implementation order

## Stop Conditions

- all retained capabilities have a local home
- all exclusions are explicit

## Quality Gate

- never force false equivalence
- name what is intentionally excluded
